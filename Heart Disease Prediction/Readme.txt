❤️ HEART DISEASE PREDICTION & ANALYSIS
===========================================================

DESCRIPTION:
This Python script performs comprehensive data cleaning, visualization, 
and machine learning to predict heart disease presence using patient data.

FEATURES:
- Data Cleaning: Handles missing values and removes duplicate entries.
- Advanced Visualization: Generates correlation heatmaps and target distributions.
- Predictive Modeling: Implements Logistic Regression with feature scaling.
- Evaluation Metrics: Provides Accuracy scores, Confusion Matrices, and ROC/AUC curves.
- Feature Importance: Ranks clinical factors by their impact on the prediction.

PREREQUISITES:
1. Python 3.8+.
2. Dataset: A 'heart.csv' file in the same directory.
3. Libraries: pandas, numpy, seaborn, matplotlib, scikit-learn.

INSTALLATION:
   pip install pandas numpy seaborn matplotlib scikit-learn

HOW TO RUN:
- Run 'python heart_analysis.py' in your terminal.
- View console for accuracy results and pop-up windows for clinical insights.
