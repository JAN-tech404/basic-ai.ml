DESCRIPTION:
This Python script performs an Exploratory Data Analysis (EDA) on the 
classic Iris flower dataset to identify patterns between species.

FEATURES:
- Data Profiling: Automated shape, column, and statistical summaries.
- Statistical Insights: Detailed distribution and outlier analysis.
- Multi-view Visualization: High-quality Pairplots, Histograms, and Boxplots.

PREREQUISITES:
1. Python 3.8 or higher.
2. Required Libraries: pandas, matplotlib, seaborn.

INSTALLATION:
Install dependencies via pip:
   pip install pandas matplotlib seaborn

HOW TO RUN:
- Script: Run 'python iris_analysis.py' in your terminal.
- Notebook: Execute all cells to generate the inline plots.

VISUALIZATIONS INCLUDED:
- Pairplot: Relationships between all features color-coded by species.
- Histplot (KDE): Distribution density of sepal length.
- Boxplot: Statistical spread and median values per species.
